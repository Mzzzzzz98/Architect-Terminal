## 🚀 什么是 Nebula Terminal？

在信息爆炸的时代，我们的注意力正在被无限切割。**Nebula Terminal** 是一款基于 Python 构建的桌面端沉浸式效率枢纽。它融合了双模状态切换、全息光效引擎、第二大脑双链笔记，以及硬核的“猎杀协议”，旨在为你打造一个纯粹的数字禅修室与游戏指挥舱。


## ✨ 核心特性 (Core Features)

### 🌓 双子星模式 (Dual-Mode Matrix)
* **STUDY 模式 (赛博禅修)**：极简专注界面，环境雷达、每日金句与每周算力进度条，为你过滤一切外界干扰。
* **GAME 模式 (战斗指挥舱)**：暴躁战斗红主题，一键启动你最爱的 3 款 3A 大作，接管你的娱乐时光。

### 🎨 霓虹光效引擎 (Neon Resonance Engine)
* **毫秒级热重载**：内置极客调色盘，支持为不同模式自定义霓虹 Accent Color。保存瞬间，全界面（文字、进度条、猫咪轮廓）光效同步跃迁，无需重启。
* **赛博电子宠 (Cyber-Pet)**：悬浮窗形态下的智能伴侣。它的眼睛色彩与情绪（Normal/Tired/Sleepy/Angry）会根据你的专注或游戏时长实时演化。支持物理级“撸猫”互动与仿生反馈。

### 🛡️ 猎杀协议 (Hunt Protocol)
* **黑名单精准狙击**：一键扫描系统内存，精准锁定高耗能/易分心的干扰进程（如微信、浏览器、游戏平台）。
* **温和劝退机制**：摒弃暴力的进程查杀，采用 `Graceful Close` 信号，让目标软件体面保存数据并退出，彻底杜绝系统崩溃风险。

### 🧠 第二大脑双链引擎 (Second Brain & Gallery)
* **Roam 级双向链接**：在日记中输入 `[[关键词]]`，即可自动渲染为赛博高亮链接。点击瞬间拉起全局思想矩阵，精准追溯历史记忆锚点。
* **全息图库 (Asset Gallery)**：内置原生剪贴板接管技术，支持直接粘贴图片并以瀑布流网格展示。支持一键调起外部查看器与资产安全导出。

## 🛠️ 安装与运行 (Installation)

### 方案 A：直接使用安装包 (推荐)
1. 前往 [Releases](你的GitHub发布页面链接) 页面。
2. 下载最新的 `Nebula_Setup_v1.0.0.exe`。
3. 双击运行，跟随安装向导完成安装。

### 方案 B：源码编译运行 (极客专属)
如果你想从零开始掌控源码，请确保你的系统已安装 Python 3.10+。

```bash
# 1. 克隆星云终端矩阵
git clone [https://github.com/你的用户名/Nebula-Terminal.git](https://github.com/你的用户名/Nebula-Terminal.git)
cd Nebula-Terminal

# 2. 安装底层依赖引擎
pip install -r requirements.txt

# 3. 点火起飞
python main.py
